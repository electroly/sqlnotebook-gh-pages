System Requirements for Windows 7

SQL Notebook supports Windows 7, but the following updates must be installed:

  • Service Pack 1 - Installation instructions
  • Windows Update KB3063858 - Download the updater for 64-bit • 32-bit
If the SQL Notebook window does not appear when you launch the application, make sure that you have KB2533623 installed. Click the link above to download and install the update, then try launching SQL Notebook again.
